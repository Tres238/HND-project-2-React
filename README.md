# 🎟️ TicketEase - React Version

This is the **React** version of the TicketEase project — a simple ticket management web app that allows users to log in, view dashboards, and manage tickets easily.

---

## 🚀 Live Demo
👉 [View TicketEase on Vercel](https://your-vercel-link.vercel.app/

## 🛠️ Features
- User login simulation  
- Dashboard with ticket stats  
- Ticket management section  
- Clean and responsive design  
- Reusable React components (Navbar, Footer, Dashboard, etc.)

---

## ⚙️ Technologies Used
- React (JSX Components)
- CSS for styling
- Vercel for hosting
- GitHub for version control
